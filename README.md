# Análisis de Popularidad Pokémon  

Este repositorio corresponde a la primera entrega del curso Data Science 2 (Coderhouse).

El objetivo del proyecto es analizar qué variables influyen en la popularidad de los Pokémon mediante la construcción de un índice cuantitativo de popularidad.

Para ello, se combinan:

- Datos estructurales del dataset de Kaggle  
- Información adicional obtenida desde PokeAPI  
- Interés de búsqueda extraído de Google Trends  

El enfoque es exploratorio y busca identificar patrones, relaciones y posibles factores determinantes.



## Contenido del repositorio

- Notebook en Google Colab (`.ipynb`) con el análisis exploratorio completo  
- Presentación en formato PDF/PPT utilizada para la entrega virtual  
- Visualizaciones y validación de hipótesis  



## Hipótesis evaluadas

1. Relación entre poder total y popularidad.  
2. Impacto del tipo elemental en la popularidad.  
3. Influencia de la generación (efecto nostalgia).  
4. Diferencia entre Pokémon legendarios y no legendarios.  



## Tecnologías utilizadas

- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Pytrends  
- Requests (PokeAPI)  



## Estado del proyecto

Primera fase: Análisis Exploratorio de Datos (EDA).  

La siguiente etapa incluirá el desarrollo de un modelo predictivo para estimar la popularidad a partir de las variables analizadas.
